Industry News for Progressive Web Apps (PWA).This is a simple news site displaying links and summary information for public articles on the topic. The site uses Microsoft's Bing Search API to search for articles using the keyword 'PWA', and renders the top 20 articles below. This site is one of the sample apps from Learning Progressive Web Apps; it's a PWA and supports installation, data caching, and more. Check out the book for more information.
